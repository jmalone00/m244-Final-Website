# Statistical Analysis of the 2022-23 NBA MVP Race
Authors: John Malone and Kreekor Karageuzian

Course: MATH 244

### Project Overview
This project analyzes the 2022-2023 NBA MVP race using statistical modeling, EDA, logistic
regression, and clustering methods. We developed a custom MVP score using production stats
and efficient metrics to compare player performance against actual MVP voting results.  

### Project Website
Website link: https://jmalone00.github.io/m244-Final-Website/

### Data Sources
- NBA Stuffer Player Props Dataset
- Kaggle Advanced Statistics Dataset

### Methods Used
- Z-score standardization
- Custom MVP scoring model
- Logistic Regression
- Clustering

### Main Findings
Our custom MVP model ranks Nikola Jokic higher than Joel Embiid as the strongest MVP candidate
based on their production stats, efficiency metrics, and team success. The logistic regression
model results show that higher MVP scores were strongly associated with MVP candidacy.
